## Get started with [Theme Builder](https://explysm.github.io/revenge-things/theme-builder)

> Create a new github repo to host your themes (or you could use a different hosting site.)
>
> Go to https://explysm.github.io/revenge-things/theme-builder
>
> Create your theme
>
> Upload your theme to your github repo (or whatever hosting you used)
>
> copy the raw file link
>
> go to "themes" setting in revenge, and paste the link
